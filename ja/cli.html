CLI¶

Symbol CLI is an open-source command-line interface to interact with the blockchain.

Installation¶

Symbol CLI is distributed using the node package manager npm.

注釈

Symbol CLI require Node.js 12 LTS to execute.

Execute the following command in a new terminal:

npm install --global symbol-cli

Configuration¶

To start using Symbol CLI, configure a profile. Profiles are used to set a base URL and account to sign transactions.

symbol-cli profile import --private-key <PRIVATE_KEY> --network TEST_NET --url http://api-01.ap-northeast-1.testnet-0951-v1.symboldev.network:3000 --password <PASSWORD> --profile <PROFILE_NAME>

By default, Symbol CLI will always use the default profile. To use a named profile with any other command, add the --profile option to the command.

symbol-cli account info --profile <PROFILE_NAME>

If you are going to use named profile for multiple commands, you can change the default profile with the following command.

symbol-cli profile setdefault --profile <PROFILE_NAME>

If you do not have a private key to create a profile, you can generate a new account, add a node URL and save it as default or named profile.

symbol-cli account generate --network TEST_NET --url http://api-01.ap-northeast-1.testnet-0951-v1.symboldev.network:3000 --password <PASSWORD> --profile <PROFILE_NAME> --save

Guides¶

  • ワークステーションのセットアップ

    最初のガイドでは Symbol 開発を始めるために必要なツールのインストールをステップバイステップに進めていきます。

  • アカウントの作成

    トランザクションの送信と受信をするために、新しい Symbol アカウントを作成します。

  • アカウントの情報を取得する

    公開鍵とアカウントの残高を取得します。

  • モザイクとメッセージを２つのアカウント間で送信

    定義して、署名したら転送トランザクションをアナウンスします。

  • ネームスペースの取得

    アセットを識別する、一意の名前を作成します。

  • ブロック高によるブロックの取得

    与えられたブロック高のブロック情報を取得する。

  • モザイクの作成

    モザイクを使用した資産のトークン化

  • アグリゲートボンデッドトランザクションの連署

    アグリゲートボンデッドトランザクションに署名する方法

  • サブネームスペースの作成

    ルート ネームスペース を取得すると 3 つまでのサブネームスペースを作って、アセットの整理をすることができます。

  • トランザクションをアカウントから読み込む

    アカウントが関連するトランザクション一覧の取得

  • モザイク供給量の変更

    「Supply Mutable」 プロパティを使用してモザイクを作成した場合、トークンを増やすか、総供給量を減らすことができます。

  • 新しいブロックの監視

    新しいブロックが取り込まれた通知を受け取る。

  • ネームスペースをアドレスへリンクする

    トランザクションを発行するとき、他の人がよりわかりやすい方法でアカウントを参照できるように、ネームスペースをアドレスにエイリアスとして設定します。

  • ネームスペースをモザイクへリンクする

    トランザクションを発行するとき、他の人がよりわかりやすい方法でモザイクを参照できるように、ネームスペースをモザイクにエイリアスとして設定します。

  • ネームスペースの情報を取得する

    ネームスペース識別子による所有権と期間の取得

  • モザイクの情報を取得する

    モザイク識別子による所有権、可分性、期間、フラグの取得

  • ネームスペース登録期限の延長

    このガイドでは、ネームスペース期限の延長の仕方を示します。

  • アカウントにメタデータを割り当てる

    アカウントにカスタムデータを追加

  • ネームスペースにメタデータを割り当てる

    ネームスペースにカスタムデータを追加

  • モザイクにメタデータを割り当てる

    モザイクにカスタムデータを追加

  • アカウントにアタッチされたメタデータエントリを取得する

    アカウントにアタッチされたメタデータエントリの一覧を取得する
