転送トランザクション¶

転送トランザクションは モザイク とメッセージを2つの アカウント 間で送信するために使われます。

Alice は 10 symbol.xym を Bob へ送る

受信者¶

受信者は転送トランザクションを受け取る アカウント のアドレスです。

過去にトランザクションに関与したことがなくても、正当なアドレスであればモザイクを送信することができます。

注釈

もし受信するアドレスの秘密鍵を誰も持っていない場合、その資産は永久に使うことができなくなります。

モザイク¶

モザイク はトークンといえますが、報酬ポイント、株式、署名、状態フラグ、投票やその他の貨幣などのより特化した資産とみなすこともできます。

同じトランザクションで異なるモザイクの組み合わせを送信することができます。

メッセージ¶

パブリックネットワーク では、転送トランザクションは 1023 文字までの長さのメッセージを持つことができ、ブロックチェーンにデータを永続的にタイムスタンプするのに適しています。

デフォルトでは、添付されたメッセージはネットワーク参加者の全員が見ることができます。

暗号化メッセージ¶

暗号化メッセージは送信者と受信者だけが平文を読むことができます。

symbol.xym は CBC モード の Bouncy Castle’s AES ブロック暗号化実装を使ってメッセージの暗号化と復号を行います。

注釈

crypto モジュールの下に暗号化メッセージの エンコード と デコード の方法を発見することができますが、替わりに SDK のパブリックメソッド を使用することを推奨します。

ガイド¶

  • モザイクとメッセージを２つのアカウント間で送信

    定義して、署名したら転送トランザクションをアナウンスします。

  • 暗号化メッセージの送信

    受信者アカウントでしか読めない暗号化メッセージを送信します。

トランザクションスキーマ¶

TransferTransaction¶

モザイク またはメッセージを2つの アカウント 間で送信するためには、転送トランザクションをアナウンスします。

Version: 0x01

EntityType: 0x4154

インライン

  • Transaction または EmbeddedTransaction
プロパティ タイプ 説明
recipientAddress UnresolvedAddress トランザクションレシート
mosaicsCount uint8 添付されたモザイクの数
messageSize uint16 添付されたメッセージのサイズ
transferTransactionBody_Reserved1 uint32 モザイクを 8 バイト境界に揃えた予約パディング
mosaics array(UnresolvedMosaic, mosaicsCount) 送信する添付済みモザイク
message array(byte, messageSize) メッセージタイプ と 16 進数のペイロード

MessageType¶

Enumeration: uint8

Id 説明
0x00 平文メッセージ
0x01 暗号化メッセージ
0xFE 永続的なハーベスティングのデリゲート
