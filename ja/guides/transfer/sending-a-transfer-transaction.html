モザイクとメッセージを２つのアカウント間で送信¶

定義して、署名したら転送トランザクションをアナウンスします。

ここでは、あなたのアカウントから Bob のアドレス TBONKW-COWBZY-ZB2I5J-D3LSDB-QVBYHB-757VN3-SKPP へ 10 symbol.xym を送る方法をガイドします。

前提条件¶

  • 入門セクション を完了している
  • 新しい アカウント を作成します。
  • アカウントに手数料を支払うために十分な symbol.xym を入金してください。

方法 #01: デスクトップウォレットを使用する¶

  1. デスクトップウォレットのホームから 転送 タブをクリックします。

2. Fill out the necessary information for the transfer transaction. For this example, you need to specify that you are sending 10 XYM to Bob (TCQSO3-LUEWJZ-X4ITOY-4YWVL5-TAOEJ5-6YXUMS-AJHH). You can add a message, but it is not necessary in this case.

3. Once you have filled out all the information, click 「Send」. A popup will show. Read and verify the information, then enter your wallet password and click 「Confirm」.

4. You can verify that the transaction was successful by going back to the 「Dashboard」 tab. At first, it might show up under 「Unconfirmed」 transactions as the transaction becomes included in a block, but you should soon be able to see it under the 「Confirmed」 transactions.

方法 #02: SDK を使用する¶

  1. 新しいターミナルでは、あなたのアカウントのアドレスに関係するトランザクションの承認済みとそれらのネットワークによる却下を監視します。
symbol-cli monitor all --address <YOUR-ADDRESS>
  1. Bob のアドレスが受信者で 10 symbol.xym を添付した TransferTransaction を定義してください。

すでにお気づきだと思いますが、転送トランザクションではモザイクの配列をパラメータとして必要とします。これは一度で複数のモザイクを添付したトランスファートランザクションを送ることができます。

複数のモザイクを所有している場合は、同じトランザクションでそれらを一緒に送信できます:

注釈

Symbol は主に絶対量を扱います。絶対量を取得するには、送ってほしいアセットの量に 10divisibility を乗じてください。例えば、モザイクが 可分性 2 である場合、10 単位 (相対) 送信するには 1000 (絶対) と定義します。

3. Sign the transaction with your account. Include the network generation hash to make the transaction only valid for your network. Open nodeUrl + '/node/info' in a new browser tab and copy the meta.networkGenerationHash value.

5. Open the terminal where you are monitoring the transaction’s status. The transaction should appear as confirmed after ±15 seconds and the amount defined gets transferred from the sender’s account to the recipient’s account. If the terminal raises an error, you can check the error code description here.

方法 #03: CLI を使用する¶

ターミナルを開き、あなたのデフォルトアカウントから Bob のアドレスへ 10 XYM を送るために、次のコマンドを実行します。

View Code¶
 symbol-cli transaction transfer --recipient-address TBONKW-COWBZY-ZB2I5J-D3LSDB-QVBYHB-757VN3-SKPP --mosaics "@symbol.xym::10000000" --message "This is a test message" --sync
